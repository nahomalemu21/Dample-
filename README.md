# Dental Clinic Demo Website

A Vite + React + Tailwind demo website for a dental clinic. Includes English/Amharic language switching, services, doctors, gallery, pricing, testimonials, FAQ, and booking CTAs.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Edit clinic information

Open `src/App.jsx` and update the `clinic` object near the top.
